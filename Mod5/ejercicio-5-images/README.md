# Ejercicio 5: Imágenes Responsive

## Objetivo

Implementar imágenes responsive usando picture, srcset y técnicas modernas.

## Requisitos

- Picture element
- Srcset con diferentes tamaños
- Art direction
- Imágenes de fondo responsive
- Lazy loading

## Conceptos

- `<picture>` element
- `srcset` y `sizes` attributes
- CSS background-image responsive
- loading="lazy"

## Tiempo Estimado

50 minutos
