# Ejercicio 1: Landing Page Responsive

## Objetivo

Crear una landing page completamente responsive que se adapte a móvil, tablet y desktop usando media queries y diseño mobile-first.

## Descripción

- Hero section full-screen
- Sección de características (3 columnas → 1 columna en móvil)
- Galería de imágenes responsive
- Formulario de contacto
- Footer con enlaces

## Conceptos Clave

- Mobile-first approach
- Media queries
- Viewport units(vw, vh)
- Imágenes responsive
- Breakpoints estándar

## Tiempo Estimado

90-120 minutos
