# Ejercicio 4: Tipografía Responsive

## Objetivo

Crear un sistema tipográfico completamente responsive usando clamp() y unidades fluidas.

## Requisitos

- Headings escalables
- Body text fluido
- Line-height adaptable
- Fluid typography con clamp()
- CSS custom properties

## Conceptos

- `clamp(min, preferred, max)`
- Fluid typography
- CSS variables
- rem units

## Tiempo Estimado

45 minutos
