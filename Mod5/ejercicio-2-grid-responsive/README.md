# Ejercicio 2: Grid Responsive con Auto-fit

## Objetivo

Crear una galería responsive que se adapte automáticamente usando CSS Grid y la función auto-fit.

## Requisitos

- Grid con auto-fit y minmax
- Mínimo 12 elementos
- Breakpoints personalizados
- Gap responsive
- Hover effects

## Conceptos

- `repeat(auto-fit, minmax())`
- CSS Grid responsive
- Media queries
- Responsive images

## Tiempo Estimado

45-60 minutos
