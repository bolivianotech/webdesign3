# Ejercicio 3: Navbar Responsive

## Objetivo

Crear una navbar completamente responsive con menú hamburguesa en móviles.

## Requisitos

- Logo adaptable
- Menú horizontal en desktop
- Menú hamburguesa en móvil
- Transiciones suaves
- JavaScript para toggle

## Conceptos

- Flexbox responsive
- Media queries
- CSS transitions
- Mobile navigation patterns

## Tiempo Estimado

60 minutos
